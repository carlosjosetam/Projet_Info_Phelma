Ce programme a pour but dans un premier temps l'analyse lexicale d'un fichier texte contenant du code assembleur pour un processeur MIPS.

Le fichier main.c contient le code permettant l'analyse lexicale des fichiers de teste ainsi que l'affichage du resultat dans le terminal.
Le dossier tests contient le fichier de testes: ass.s, miam.s et mult.s.
Le dossier src contient les dosiers analyse_lexicale et structures ou se trouve toutes les fichiers .c et .h nécessaire à main.c. Les fonctions sont dans le dossier analyse_lexicale, les definitions de stuctures sont dans le dossier structure.
Le fichier compiler.sh contient les instructions pour compiler et lancer le programme main.c.

Pour compiler et executer le programme, tappez "sh compiler.sh" dans le terminal.

Tous les fichiers peuvent être trouvés sur internet à l'adresse suivante:
